# Battleship

## Build Setup

### Frontend
Don't forget to install and run the [Frontend](https://github.com/redidizzy/battleship)
### Backend
All you have to do is 
```
$ yarn install
# or (if you're using npm)
$ npm run install


$ node server.js
```
## Configuration
Backend is run on port 3001 by default, to change this behaviour, dotenv is used here, so all you have to do is create a ".env" file and add your PORT environment variable
```
PORT= <YOUR_NEW_PORT>
```
